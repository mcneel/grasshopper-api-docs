Grasshopper Namespace

Top level namespace for the Grasshopper plug-in.
Classes
  ClassDescription
CentralSettings
Represents a collection of program wide settings and events for when those settings change. This should probably be the bottleneck for all program wide settings, but stuff is still littered about for now.
DataTreeT
Implements basic Grasshopper Data Tree functionality in an easy-to-use class. This class is used primarily in Scripting components. If you're using the Grasshopper SDK you should consider using the GH_StructureT class instead.
